How Percentage Decrease Works

Percentage Decrease Formula

Percentage Decrease = [(Original Value - New Value) √∑ Original Value] √ó 100

Step-by-Step Calculation

To calculate percentage decrease:

  1. Find the decrease amount: Original Value - New Value
  2. Divide the decrease by the Original Value
  3. Multiply by 100 to get the percentage

Example: Price Decrease

Original Price: $200

New Price: $160

Calculation:

1. Decrease amount: $200 - $160 = $40

2. Divide: $40 √∑ $200 = 0.2

3. Multiply: 0.2 √ó 100 = 20% decrease
