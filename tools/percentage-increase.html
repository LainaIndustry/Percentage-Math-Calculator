How Percentage Increase Works

Percentage Increase Formula

Percentage Increase = [(New Value - Original Value) √∑ Original Value] √ó 100

Step-by-Step Calculation

To calculate percentage increase:

  1. Find the difference: New Value - Original Value
  2. Divide the difference by the Original Value
  3. Multiply by 100 to get the percentage

Example: Salary Increase

Original Salary: $50,000

New Salary: $55,000

Calculation:

1. Difference: $55,000 - $50,000 = $5,000

2. Divide: $5,000 √∑ $50,000 = 0.1

3. Multiply: 0.1 √ó 100 = 10% increase

Percentage Increase FAQs

Absolute Increase: The actual numerical difference (e.g., $50,000 to $55,000 = $5,000 increase).

Percentage Increase: The relative change as a percentage (e.g., $50,000 to $55,000 = 10% increase).

Percentage is better for comparing different scales.

CAGR = [(Ending Value √∑ Beginning Value)^(1 √∑ Number of Years)] - 1

Example: Investment grows from $1,000 to $1,500 over 3 years:

CAGR = [(1500 √∑ 1000)^(1/3)] - 1 = 14.47% per year

If original value is zero, percentage increase is undefined (division by zero).

In such cases, you can only calculate absolute increase.

Example: Starting from 0 customers to 100 customers = infinite percentage increase.

Use formula: Original Value = New Value √∑ (1 + Percentage Increase/100)

Example: New value is $110 after 10% increase:

Original = $110 √∑ (1 + 10/100) = $110 √∑ 1.1 = $100
